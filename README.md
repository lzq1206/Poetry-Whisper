# Poetry Whisper

静态 GitHub Pages 网站：为近体诗与宋词提供逐字格律辅助、平仄提示、常见拗格提醒和同义雅词替换。

## 现在可用

- 唐诗：五绝、七绝、五律、七律，各含平起/仄起、首句入韵/不入韵正格。
- 宋词：忆江南、如梦令、卜算子、浣溪沙、蝶恋花、江城子、水调歌头、念奴娇的标注谱式。
- 平仄：以内置的平水韵取向字库优先判定；常见古仄今平字进入仄声表，多音字可按当前位置覆写。
- 校验：逐字合谱/不合谱、未知字、三平尾、常见“孤平拗救”提示。
- 右栏：按照当前行与已选字位给出古雅同义单字；合谱候选可直接替换。

## 重要边界

平仄和词谱存在异读、通押、正变体与具体作品传统。本网站只把明确载入的谱式当作判定依据，未知字不会假装给出确定结论。`data.js` 中保留了可扩展的词牌检索库；未载入机读正谱的词牌不会在编辑器里伪称“已精确校验”。后续应从《钦定词谱》《词律》或有明确开放许可的校勘数据逐牌补入 `forms.ci`。

## 本地预览

无需构建。直接打开 `index.html`，或在目录中执行：

```bash
python3 -m http.server 8000
```

## 部署

推送到 GitHub 默认分支后，Actions 会发布至 GitHub Pages。

## 规则资料

- 近体诗平仄、孤平与拗救：王力《诗词格律》《汉语诗律学》；实现采用保守提示，不将争议变格直接判死。
- 词谱：龙榆生《唐宋词格律》、王力《诗词格律》及《钦定词谱》。

本项目仅用于创作辅助；正式参赛、发表或唱和，请自行复核韵书与所依词谱。
